# Lasso

### Deployed App Link
https://lasso-4152.herokuapp.com/

### Run program and tests
```
cd SaaS-Lasso
# Install gems
bundle install

# Initialize the database
bin/rake db:drop:all
bin/rake db:migrate
bin/rake db:seed

# Launch local server on localhost
bin/rails server

```
